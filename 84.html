Membuat nasi goreng yang enak seperti yang dijual abang2 itu kuncinya adalah masak nasi dengan api besar dan diaduk2 dengan cepat agar tidak gosong. Kecilkan api ketika memasukkan bumbu atau bahan lain kemudian besarkan kembali api untuk mengaduk2nya sampai rata. Dengan teknik seperti itu, bahkan jika membuat nasi goreng hanya dengan margarin saja tanpa duo bawang, rasanya tetap enak. Apalagi jika pakai bumbu komplit. Yuk cobain..
